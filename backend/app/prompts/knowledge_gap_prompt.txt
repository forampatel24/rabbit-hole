Analyze the knowledge gap between known concepts and a target topic.

Target topic: {target_topic}
Known concepts: [{known_concepts}]

Identify which known concepts are relevant prerequisites for the target topic, what concepts are missing between known and target, and generate a recommended learning path from known to target.

Return ONLY valid JSON. No markdown, no explanations, no additional text.

The JSON structure MUST be:

{
  "known": ["concept1", "concept2"],
  "missing": ["missing_concept1", "missing_concept2"],
  "learning_path": ["step1", "step2", "step3"]
}

CRITICAL REQUIREMENTS:
1. known: only include concepts from the input that are directly relevant as prerequisites for the target topic
2. missing: ordered list of missing concepts from simplest/prerequisite to most advanced, needed to bridge from known concepts to the target topic (3-8 concepts)
3. learning_path: complete ordered path from the FIRST thing to learn through to the target topic. Each step should build logically on the previous one. The last step should always be the target topic.
4. The learning path should represent the shortest meaningful route from known to target
5. All fields must be arrays of strings
6. Return ONLY the JSON object - no markdown fences, no explanations
